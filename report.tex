\abstract{In this report I explore the avoidance of stuttering and glitches in
functional reactive programming systems with an implementation in the pure
functional language Haskell. I contrast this with other models of functional
reactive programming in other languages and show how capturing the inputs of a
system as well as the outputs are critical to the avoidance of stuttering.}
